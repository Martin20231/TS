# Archiv

Experimente und pausierte Projekte — **nicht** für den täglichen Radar-Betrieb.

## stellwerk/

Versuch, Signale in Train Simulator (Berlin JWD / Adlershof) per Web-Karte zu steuern.

- Lua-Signal-Scripts, Szenario `ScenarioScript.lua`
- Installer (`SETUP_ALLES.bat`, `restore_vt_hl_hs.bat`, …)
- Anleitungen: `ANLEITUNG.md`, `SIGNAL_SZENARIO.md`

**Hinweis:** Signal-Steuerung war instabil (Editor-Abstürze, Lampe reagierte nicht). Vor weiteren Tests: `restore_vt_hl_hs.bat` ausführen.

## stellwerk-bridge/

Python-Seite der Spiel-Brücke (Hupe, Tastatur, UI-Klicks, RailDriver-Bus).

- `start_stellwerk_sim.bat` — lokaler Test mit Stellwerk-Modus auf der Karte (benötigt alten Server-Stand)

## unity/

Ältere Unity/C#-Skripte (`TrainController`, Netzwerk) — nicht mit dem Radar verbunden.
