Stellwerk – nur 3 Schritte im Editor

Alles andere ist schon installiert. Du musst nur noch:

Schritt 1 – Signal in der STRECKE (Strecken-Editor)

Werkzeuge → Strecken-Editor → Berlin JWD → dein Signal anklicken.

Rechts zwei Felder eintragen:

  • Object Name = SW_S1 (hast du schon)
  • Script = TS Stellwerk SW_S1.lua ← OHNE DAS ÄNDERT SICH NICHTS!

Das Script-Feld steht oft direkt neben der Zahl 566 oder unter „Script“ / „Lua“.

STRECKE SPEICHERN (Strg+S) – nicht nur das Szenario!

Schritt 2 – Fahrplan (einmalig, wichtig!)

Oben Fahrplan → beim Spieler-Dienst:

  • Neue Zeile: Warten → 1 Sekunde
  • Feld Ereignis auslösen / Trigger Event = StellwerkStart
  • Nicht „Success Event“ oder „Bedingung“ – nur Trigger Event!

Nach dem Start muss oben „Stellwerk aktiv“ erscheinen. Sonst Fahrplan prüfen.

Einmal stellwerk\patch_fresh_start.bat ausführen, dann Szenario neu starten.

Schritt 3 – Script kompilieren (nach jedem Update)

Script → Kompilieren → Strg+S speichern.

Im Spiel

Menü oben rechts: Hp0 / Sh1 / Hp1 anklicken.

Oder: radar\start_stellwerk_sim.bat → Browser → SW_S1 anklicken.

Unten rechts muss stehen: Stellwerk OK (566/SW_S1)

Wenn Meldung kommt aber das Signal gleich bleibt: Script-Feld in Schritt 1 fehlt oder Strecke nicht gespeichert.
