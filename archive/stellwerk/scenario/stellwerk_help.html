Stellwerk Adlershof – Bedienung

Steuerung: Im Spiel öffnet sich ein HTML-Menü (klickbare Links: Hp0 / Sh1 / Hp1).

Oder Web-Karte: start_stellwerk_sim.bat → Signal SW_S1 anklicken.

  • Hp0 = Halt (rot)
  • Sh1 = Rangierfahrt (gelb auf der Karte)
  • Hp1 = Fahrt frei (weiß / grün auf der Karte)

Weichen: Taste G / Shift+G oder Klick auf der 2D-Karte (Taste 9).

Kamera: Taste 7 = Draufsicht (empfohlen für Stellwerk).

Radar-Karte: Button Stellwerk → Signal auf der Karte anklicken → Hp0 / Sh1 / Hp1. Der Tracker leitet den Befehl ins Spiel.
